# Projeto - SmartCards
SmartCards é uma aplicação web para organização de estudos com flashcards interativos. Permite criar assuntos, cadastrar perguntas e respostas, estudar de forma dinâmica e acompanhar a evolução por meio de gráficos. Utiliza HTML, CSS e JavaScript no front-end, integração com API REST e banco de dados relacional para armazenar as informações.
